### 11 手持端基本 ###

#### 11.1 检查版本号 ####

###### 接口地址 ######

[http://app.qc-wbo.com/v1/clientVersions](http://app.qc-wbo.com/v1/clientVersions)

###### 请求方法 ######
GET


###### 请求参数 ######

无

###### 响应参数 ######

|名称|类型|描述|示例值|
|---|---|---|---|
| id | Long | 主键ID | 1 |
| versionCode | Int | 版本号 | 1 |
| versionName | String(50) | 版本名称 | 1.0 |
| text | String | 说明 | 更新更新 |
| isForce | Boolean | 是否强制更新 | false |

###### 响应示例 ######

```json
{
    "id": 1,
    "versionCode": 1,
    "versionName": "1.0",
    "text": "更新更新",
    "isForce": false
}
```

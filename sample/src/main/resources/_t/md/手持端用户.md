### 4 手持端用户

#### <a href='#4.1登录' name='4.1登录'>4.1 登录</a>

###### 接口地址

POST

###### 请求参数

|---|---|---|---|---|---|---|
| jobNumber | string | 是 | 100 | 工号 |  | KD001 |

| needAddInfo | boolean | \- | 是否需要补充资料 | false |

| createdDate | long | \- | 创建时间（毫秒数） | 1482411971918 |
| id | long | \- | 主键ID | 3 |
